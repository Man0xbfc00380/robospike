# RoboSpike

A coroutine-based rclcpp for more efficient scheduling in ROS 2. The completed installation steps will be coming soon.

> [!INFO]
> Current ROS 2 version is Dashing.

H. Li, Q. Yang, S. Ma, R. Zhao and X.
